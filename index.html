About

This small project was made with D3.js, Sankey plugin, Bootstrap and Flat-UI.

Data comes from JavaScript Developer Survey by DailyJS (2012, 2013). Single choice questions are marked with [.] and multiple choice questions with [*].

If you found a bug or have a feature request, please create an issue on GitHub or contact me directly (kdzwinel at gmail, @kdzwinel).
